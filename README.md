DBF_to_CSV
===
Convert dbf files to csv using python3 and dbfread

## Beginners Guide

If you are a total beginner to this, start here!

1. Create a venv
``` 
python -m venv myenv
```
3. Install requirements
```
pip install -r requirements.txt
```
5. run script with
```
python dbf_to_csv.py file.dbf
```
